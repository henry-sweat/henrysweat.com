# henrysweat.com

My personal site — [henrysweat.com](https://henrysweat.com)

## About

A personal website built with Next.js 16, React 19, and Tailwind CSS. Deployed on Vercel.

Features:
- Blog / writing via MDX
- Bookshelf
- Resume
- Command palette (cmdk)

## Tech Stack

- *Framework*: [Next.js 16](https://nextjs.org/) (App Router)
- *Language*: TypeScript
- *Styling*: Tailwind CSS + shadcn/ui (Radix UI)
- *Content*: MDX via next-mdx-remote
- *Deployment*: Vercel (Node 24)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Scripts

| Command | Description |
|---|---|
| `npm run dev` | Start dev server |
| `npm run build` | Production build |
| `npm run lint` | Run ESLint |
